# 📝 COME MODIFICARE UN TEMPLATE ESISTENTE

## Per MODIFICARE un template:

### 1. **Trova il template nella lista**
   - Usa la ricerca per trovarlo velocemente
   - Oppure scorri la lista

### 2. **Clicca sull'icona MATITA (✏️)**
   - Si trova sulla destra di ogni template
   - Accanto alle altre icone (occhio, check, cestino)

### 3. **Si apre l'editor con i dati precaricati**
   - Tutti i campi sono già compilati
   - Puoi modificare quello che vuoi
   - Le variabili esistenti sono già presenti

### 4. **Fai le modifiche**
   - Cambia testo, HTML, SMS, etc.
   - Aggiungi o rimuovi variabili
   - Modifica categoria o priorità

### 5. **Clicca "Aggiorna Template"**
   - Il bottone dirà "Aggiorna" invece di "Crea"
   - Le modifiche vengono salvate

---

## 🎯 ICONE NELLA LISTA TEMPLATE:

Per ogni template hai 4 icone sulla destra:

1. **✏️ MATITA** = Modifica il template
2. **👁️ OCCHIO** = Anteprima (funzione futura)
3. **✅/❌ CHECK** = Attiva/Disattiva il template
4. **🗑️ CESTINO** = Elimina (solo per template non di sistema)

---

## ⚠️ NOTA IMPORTANTE:

- I template di **SISTEMA** (con badge grigio "Sistema") possono essere modificati ma NON eliminati
- I template **CUSTOM** (creati da te) possono essere sia modificati che eliminati

---

## 🔍 SUGGERIMENTO:

Se non vedi le icone:
1. Assicurati che i template siano caricati (non deve esserci "Nessun template trovato")
2. Le icone sono sulla DESTRA di ogni box template
3. Passa il mouse sopra per vedere il tooltip con la descrizione
